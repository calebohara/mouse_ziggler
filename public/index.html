noidle.app

Stay active. Stay available.

A friendly Windows tray app that keeps the green dot in Microsoft Teams green — and your machine awake — without nagging you.

⬇ Download for Windows .msi installer

Windows 10 / 11 · single-file .exe or per-user .msi · MIT

How it works

Every ~45 seconds (configurable, randomized so it doesn't look mechanical), noidle.app does three small things:

1. A whisper of input

It nudges the cursor by one pixel and presses an invisible key the OS counts as activity. Nothing visible happens on screen.

2. Wakelock

It pins your display and system awake so they don't sleep underneath you mid-thought.

3. Self-check

It reads back the same idle counter Teams checks. If the nudge didn't take, it logs a warning instead of silently lying to you.

Polite by default

  • Smart pause — skips the nudge when you're already typing
  • Screen-share aware — pauses while you're presenting in Teams
  • Start with Windows — one-click toggle, no admin needed
  • Global hotkey — Ctrl+Alt+Z to start or pause
  • "What's new" updates — quiet popup with friendly release notes
  • Stats & logs — uptime, ticks, full log file

Honest about its limits

noidle.app keeps you Available while you're at the desk but stepped away from the mouse. It can't override:

Locked workstation · Outlook calendar status · Corporate EDR / monitoring tools · Disconnected RDP

If you need stealth against employee-monitoring software, you need hardware (a microcontroller flashed with HID firmware) — that's on the v2 roadmap.
